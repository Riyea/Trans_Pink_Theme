# Trans Pink Theme for JetBrains IDEs

A soft trans-flag pink theme for JetBrains IDEs.

This theme includes two variants:

- **Trans Pink Light**
- **Trans Pink Dark**

The dark theme is designed around a gray-pink Islands-style UI, with rounded separated panels, soft contrast, and the transgender flag pink `#F5A9B8` as the main accent color.

## Installation

1. Download `Trans_Pink_Theme.jar` from the Releases page.
2. Open your JetBrains IDE.
3. Go to `Settings / Preferences`.
4. Open `Plugins`.
5. Click the gear icon.
6. Choose `Install Plugin from Disk...`.
7. Select `Trans_Pink_Theme.jar`.
8. Restart the IDE.
9. Go to `Appearance & Behavior → Appearance → Theme`.
10. Select `Trans Pink Light` or `Trans Pink Dark`.

## Design Notes

The theme uses `#F5A9B8` as the main accent color instead of the full background color, so the interface stays readable and calm.

The dark version uses:

- top/header area: `#242528`
- main editor and project body: `#303033`
- cards and input fields: `#38383B`
- accent and focus color: `#F5A9B8`

This creates a soft gray-pink look while keeping the UI usable for long coding sessions.

## Compatibility

This theme is designed for JetBrains IDEs that support custom theme plugins.

The Islands-style rounded panels work best on newer JetBrains IDE versions with Islands Theme support.
